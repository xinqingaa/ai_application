# 课程文档模板

本文件用于后续新增或重写课程、章节、代码 README 时对齐结构。

## 阶段 README 模板

```md
# 阶段名称

## 目标

- 学完后能做什么
- 本阶段服务后续哪个能力

## 第一入口

1. 先读哪个大纲
2. 再读哪些章节
3. 如果要写代码，先进入哪个 `source/` 目录

## 章节顺序

| 顺序 | 文档 | 代码入口 | 状态 |
|------|------|----------|------|
| 1 | ... | ... | ... |

## 完成标准

- 能运行哪些命令
- 能解释哪些概念
- 能产出什么 demo 或项目
```

...

## 4. 代码映射

| 文档内容 | 代码文件 | 作用 |
|----------|----------|------|
| ... | ... | ... |

## 5. 练习

...

## 6. 完成标准

- ...
```

## 代码 README 模板

```md
# 章节或项目名称

> 本 README 说明如何运行当前代码。完整理论请看对应课程文档。

## 对应课程文档

- `../../../course/xx/xx.md`

## 当前状态

- 已完成：
- 未完成：

## 第一运行入口

```bash
python 01_demo.py
```

## 运行命令

```bash
python -m pip install -r requirements.txt
python 01_demo.py
python -m unittest discover -s tests
```

## 文件说明

| 文件 | 作用 |
|------|------|
| ... | ... |

## 完成标准

- 能运行...
- 能解释输出...
- 能通过测试...
```
