# 04 RAG Labs

`labs/` 用于存放 `04_rag` 九章学习快照。

每个目录对应一章文档：

| Phase | 对应文档 | 角色 |
|-------|----------|------|
| `phase_1_scaffold/` | `01_rag_basics.md` | 已建立的项目骨架快照 |
| `phase_2_document_processing/` | `02_document_processing.md` | 文档处理快照 |
| `phase_3_embeddings/` | `03_embeddings.md` | 向量化快照 |
| `phase_4_vector_databases/` | `04_vector_databases.md` | 向量数据库快照 |
| `phase_5_retrieval_strategies/` | `05_retrieval_strategies.md` | 检索策略快照 |
| `phase_6_rag_generation/` | `06_rag_generation.md` | RAG 生成快照 |
| `phase_7_rag_optimization/` | `07_rag_optimization.md` | 评估优化快照 |
| `phase_8_advanced_rag/` | `08_advanced_rag.md` | 进阶方向认知快照 |
| `phase_9_project_integration/` | `09_rag_project.md` | 综合项目整合快照 |

学习时优先进入对应 phase，不要一开始进入最终 `rag_lab`。
